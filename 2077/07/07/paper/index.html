Paper

  • https://github.com/RacerZ-fighting/PL-Compiler-Resource

    PL相关的知识集合

  • 面向源代码的软件漏洞静态检测综述 √
    综述类 paper

  • tabby 毕业设计论文

  • P/Taint: Unified Points-to and Taint Analysis √
    讲述 污点分析和指针分析之间的关系

  • Tai-e: A Static Analysis Framework for Java by Harnessing the Best Designs of Classics
    taie -e框架

  • Self-Inferencing Reflection Resolution for Java
    第一篇反射解决提出的paper

  • Understanding and Analyzing Java Reflection
    最近的一篇针对反射的paper

  • Taming reflection: Aiding static analysis in the presence of reflection and custom class loaders

    一篇动态分析的技术

  • 沈维军, 汤恩义, 陈振宇, 等. 数值稳定性相关漏洞隐患的自动化检测方法

    动静态结合

  • YAMAGUCHI F, GOLDE N, ARP D, et al. Modeling and discovering vulnerabilities with code property graphs[C] √

    Yamaguchi 提出代码属性图的概念，其结合了抽象语法树、控制流图和程序依赖图

  • 王蕾, 李丰, 李炼, 等. 污点分析技术的原理和实践应用

  • M. Martin, B. Livshits, and M. S. Lam. Finding application errors and security flaws using pql: Program query language. In Proc. ofACM SIGPLAN International Conference on Object-Oriented Programming, Systems, Languages and Applications (OOPSLA), 2005

    PQL语言的漏洞分析思路
